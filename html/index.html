Documentação do Projeto de Escultura 3D em C++

Introdução

O objetivo deste projeto é construir uma ferramenta em C++ que permita realizar esculturas em blocos representados por matrizes digitais, seguindo a ideia usada no jogo Minecraft. A ideia principal é possibilitar aos usuários a criação de arquivos de texto simples com a extensão .off contendo uma sequência de comandos para definir as propriedades de uma matriz tridimensional. Essas propriedades incluem informações de cor para os elementos, permitindo que os usuários visualizem e apreciem suas criações com o auxílio de softwares de visualização tridimensional.

Tipos de Formas Geométricas

Dentro do sistema de escultura tridimensional, estão previstos alguns tipos de formas geométricas simples que os usuários poderão utilizar para criar suas esculturas. Essas formas servem como blocos de construção fundamentais para a criação de estruturas complexas e detalhadas.

Exemplo de Escultura - Floresta Mágica

Neste exemplo, demonstramos a incrível capacidade de nossa ferramenta de escultura 3D ao criar uma cena pitoresca conhecida como a "Floresta Mágica". Esta composição mágica combina elementos diversos para formar um cenário encantador que mistura realismo com imaginação.

Para visualizar sua escultura em 3D, recomendamos o uso das seguintes ferramentas:

  • 3D Viewer Online: Uma ferramenta de visualização 3D online que permite que você visualize sua escultura diretamente no seu navegador.
  • MeshLab: Um software de visualização 3D que oferece recursos avançados para a visualização e edição de modelos tridimensionais. Você precisará instalá-lo para obter a melhor experiência de visualização.

A extensão abaixo é do site 3dviewer.net com o arquivo 'floresta.off' que foi gerado pelo projeto.
